## ğŸ› Troubleshooting

### Styles not showing
1. Check `glow.css` is imported in `index.css`
2. Verify Tailwind config includes correct paths
3. Rebuild: `npm run build`

### Focus states not visible
1. Use keyboard (Tab) to test
2. Check browser devtools for `:focus-visible`
3. Verify `focus-glow` class in CSS

### Responsive not working
1. Test in browser devtools responsive mode
2. Verify Tailwind breakpoints (md, lg)
3. Check viewport meta tag
